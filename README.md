# ShopEase – E-Commerce Website UI

**ShopEase** is a static e-commerce website UI designed using **HTML** and **CSS**. It demonstrates a modern and responsive layout, focusing on clean design, visual consistency, and a user-friendly interface.

## Key Features

- Responsive navigation bar with links to main sections
- Product grid showcasing items with images, titles, and prices
- Banner section for promotions and featured products
- Footer with contact information and useful links
- Clean and consistent design with proper spacing, typography, and alignment

## Tech Stack

- **HTML** – Structure of the website  
- **CSS** – Styling, layout, and responsive design
